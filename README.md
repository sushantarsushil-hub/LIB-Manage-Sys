##JAVA PROJECT
Project Name-Library management system

Features-
1. User Login & Regestration System
   Create a simple login page for librarian/admin.
   What to implement
   Username and password field
   Login button
   Simple validation
   Java concepts used
   GUI (Swing)
   Conditional statements
    Event handling
   
3. Add Book
  Allow librarian to add books.
  Book information
  Book ID
  Book Name
  Author Name
  Category
  Quantity

3. View All Books
  Display all added books in a table or text area.

4. Search Book
  Search books using:
  Book ID
  Book Name

5. Issue Book
6. Return Book
7. Delete Book
  Remove book from library list.

8. Simple Dashboard/Home Page
  After login, show buttons:
  Add Book
  View Book
  Search Book
  Issue Book
  Return Book
  Logout
